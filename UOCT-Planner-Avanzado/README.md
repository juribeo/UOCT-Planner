# UOCT Planner Avanzado

Base avanzada lista para trabajar y editar.

## Ejecutar

Desde VS Code, abre esta carpeta y en la terminal:

```powershell
cd backend
..\.venv\Scripts\activate
pip install -r requirements.txt
python -m uvicorn app.main:app --reload
```

Luego abre: http://127.0.0.1:8000
